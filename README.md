# [在爱中保有自我——终极的吸引力](https://hoo.be/yingsdr)

爱情常常伴随着一种融合的诱惑，让人渴望与对方合二为一。但无数事实证明，失去自我，是吸引力消散的开始。当你为了取悦对方而放弃自己的爱好、疏远自己的朋友、改变自己的观点时，你便交出了那个最初让他/她为之倾心的、独特的灵魂。

保有自我，意味着坚守你的核心价值观、你的热情所在和你的人生目标。它是在关系中依然能听见自己内心的声音，并勇敢地遵从它。一个有清晰自我边界的人，是富有魅力的。他/她不依赖于伴侣的评价来定义自己的价值，这种内在的稳定与自信，本身就构成了持久的吸引力。

最深沉的爱情，发生在两个坚硬的“自我”之间，是完整的个体与个体之间的共鸣与欣赏。它不是依赖，而是相互映照与扶持。你爱他，是爱那个神采飞扬地追求自己梦想的他/她。同样，你也必须成为那样的人。在爱中保有自我，不仅是对自己的忠诚，也是对你所爱之人的最高尊重，更是让爱情之树常青的、最根本的养分。
